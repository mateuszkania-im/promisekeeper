# PromiseKeeper Public Pages

This repository hosts public PromiseKeeper pages for App Store metadata links.

## Files

- `index.html` - privacy policy page
- `privacy.html` - privacy policy canonical page
- `terms.html` - terms of use page
- `eula.html` - EULA pointer page (Apple standard EULA)
- `support.html` - support page
- `marketing.html` - marketing / landing page

## GitHub Pages URL

After enabling Pages (`Settings -> Pages -> Deploy from branch -> main / root`), pages will be available at:

- Privacy Policy URL: `https://mateuszkania-im.github.io/promisekeeper/privacy.html`
- Terms URL: `https://mateuszkania-im.github.io/promisekeeper/terms.html`
- EULA URL: `https://mateuszkania-im.github.io/promisekeeper/eula.html`
- Support URL: `https://mateuszkania-im.github.io/promisekeeper/support.html`
- Marketing URL: `https://mateuszkania-im.github.io/promisekeeper/marketing.html`
